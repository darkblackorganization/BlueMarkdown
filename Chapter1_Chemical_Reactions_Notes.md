## 📌 अध्याय का परिचय

हमारे दैनिक जीवन में कई परिवर्तन होते हैं जो रासायनिक अभिक्रियाओं के उदाहरण हैं:

| घटना | प्रकार |
|------|--------|
| दूध का खट्टा होना | रासायनिक परिवर्तन |
| लोहे पर जंग लगना | रासायनिक परिवर्तन |
| अंगूर का किण्वन | रासायनिक परिवर्तन |
| भोजन पकाना | रासायनिक परिवर्तन |
| पाचन क्रिया | रासायनिक परिवर्तन |
| श्वसन | रासायनिक परिवर्तन |

> **🔑 परिभाषा:** जब कोई **रासायनिक परिवर्तन** होता है तो उसे **रासायनिक अभिक्रिया** कहते हैं।

---

## 🔍 रासायनिक अभिक्रिया होने के संकेत (Indicators)

किसी भी **चार** प्रेक्षणों से रासायनिक अभिक्रिया की पहचान होती है:

1. ✅ **अवस्था में परिवर्तन** (Change in State)
2. ✅ **रंग में परिवर्तन** (Change in Colour)
3. ✅ **गैस का निकास / उत्सर्जन** (Evolution of Gas)
4. ✅ **तापमान में परिवर्तन** (Change in Temperature)

## 📝 1.1 रासायनिक समीकरण (Chemical Equations)

### 1.1.1 शब्द-समीकरण (Word Equation)

> किसी रासायनिक अभिक्रिया को शब्दों में लिखना **शब्द-समीकरण** कहलाता है।

**उदाहरण:**
```
मैग्नीशियम + ऑक्सीजन → मैग्नीशियम ऑक्साइड   ...(1.1)
   (अभिकारक)                    (उत्पाद)
```

- **अभिकारक (Reactants):** जो पदार्थ अभिक्रिया में भाग लेते हैं → **बाईं ओर (LHS)**
- **उत्पाद (Products):** जो नए पदार्थ बनते हैं → **दाईं ओर (RHS)**
- **तीर (→):** अभिक्रिया की दिशा दर्शाता है

---

### 1.1.2 रासायनिक सूत्र से समीकरण लिखना

शब्दों की जगह रासायनिक सूत्रों का उपयोग करके समीकरण लिखा जाता है।

```
Mg + O₂ → MgO   ...(1.2)   ← यह ढाँचा/कंकाली समीकरण है (असंतुलित)
```

---

### 1.1.3 संतुलित रासायनिक समीकरण का महत्व

**द्रव्यमान संरक्षण का नियम:** किसी भी रासायनिक अभिक्रिया में द्रव्यमान का न तो निर्माण होता है, न ही विनाश।

> **इसलिए:** समीकरण के **दोनों ओर** प्रत्येक तत्व के परमाणुओं की संख्या **समान** होनी चाहिए।

#### ✅ संतुलित समीकरण का उदाहरण:

```
Zn + H₂SO₄ → ZnSO₄ + H₂   ...(1.3)
```

| तत्व | LHS (अभिकारक) | RHS (उत्पाद) |
|------|--------------|-------------|
| Zn   | 1            | 1           |
| H    | 2            | 2           |
| S    | 1            | 1           |
| O    | 4            | 4           |

✅ **यह संतुलित है।**

---

### 1.1.4 समीकरण संतुलित करना — हिट एंड ट्रायल विधि

**उदाहरण:** `Fe + H₂O → Fe₃O₄ + H₂`

**चरण 1:** प्रत्येक सूत्र को बॉक्स में रखें (बॉक्स के अंदर कुछ न बदलें)

**चरण 2:** परमाणुओं की गिनती करें (असंतुलित):

| तत्व | LHS | RHS |
|------|-----|-----|
| Fe   | 1   | 3   |
| H    | 2   | 2   |
| O    | 1   | 4   |

**चरण 3:** सबसे अधिक परमाणु वाले यौगिक से शुरू करें → **Fe₃O₄** चुनें → **O** संतुलित करें
```
Fe + 4H₂O → Fe₃O₄ + H₂   ...(1.6)
```

**चरण 4:** H संतुलित करें:
```
Fe + 4H₂O → Fe₃O₄ + 4H₂   ...(1.7)
```

**चरण 5:** Fe संतुलित करें:
```
3Fe + 4H₂O → Fe₃O₄ + 4H₂   ...(1.8)
```

**चरण 6:** जाँच:

| तत्व | LHS | RHS |
|------|-----|-----|
| Fe   | 3   | 3   |
| H    | 8   | 8   |
| O    | 4   | 4   |

✅ **संतुलित समीकरण:**
```
3Fe(s) + 4H₂O(g) → Fe₃O₄(s) + 4H₂(g)   ...(1.10)
```

---

### 1.1.5 भौतिक अवस्थाओं के संकेत

| संकेत | अर्थ |
|-------|------|
| **(s)** | ठोस (Solid) |
| **(l)** | द्रव (Liquid) |
| **(g)** | गैस (Gas) |
| **(aq)** | जलीय विलयन (Aqueous Solution) |

> **Note:** कभी-कभी ताप, दाब, उत्प्रेरक को तीर के ऊपर/नीचे दर्शाया जाता है।

## ⚗️ 1.2 रासायनिक अभिक्रियाओं के प्रकार

---

### 🔹 1.2.1 संयोजन अभिक्रिया (Combination Reaction)

> **परिभाषा:** जब **दो या दो से अधिक पदार्थ** मिलकर **एकल उत्पाद** बनाते हैं।

**सामान्य रूप:** `A + B → AB`

| उदाहरण | समीकरण |
|---------|--------|
| CaO + जल | `CaO(s) + H₂O(l) → Ca(OH)₂(aq) + ऊष्मा` |
| कोयले का दहन | `C(s) + O₂(g) → CO₂(g)` |
| जल का निर्माण | `2H₂(g) + O₂(g) → 2H₂O(l)` |

> **⚠️ ध्यान दें:** CaO = बिना बुझा चूना (Quick Lime), Ca(OH)₂ = बुझा हुआ चूना (Slaked Lime)

**रोचक तथ्य:** Ca(OH)₂ + CO₂ → CaCO₃ + H₂O — यही दीवार पर सफेदी करने के 2-3 दिन बाद चमक आने का कारण है। संगमरमर का सूत्र भी CaCO₃ है।

---

### 🔥 ऊष्माक्षेपी अभिक्रिया (Exothermic Reaction)

> **परिभाषा:** जिन अभिक्रियाओं में उत्पाद के साथ **ऊष्मा भी उत्पन्न होती** है।

**उदाहरण:**
- प्राकृतिक गैस का दहन: `CH₄(g) + 2O₂(g) → CO₂(g) + 2H₂O(g) + ऊर्जा`
- **श्वसन (Respiration):** `C₆H₁₂O₆(aq) + 6O₂(aq) → 6CO₂(aq) + 6H₂O(l) + ऊर्जा`
- शाक-सब्जियों का कंपोस्ट बनना
- CaO + H₂O → Ca(OH)₂ + ऊष्मा

---

### 🔹 1.2.2 वियोजन / अपघटन अभिक्रिया (Decomposition Reaction)

> **परिभाषा:** जिसमें **एकल पदार्थ टूटकर** दो या दो से अधिक पदार्थ देता है। (संयोजन का विपरीत)

**सामान्य रूप:** `AB → A + B`

#### प्रकार के आधार पर वियोजन:

| प्रकार | ऊर्जा का स्रोत | उदाहरण |
|--------|---------------|--------|
| **ऊष्मीय वियोजन** | ऊष्मा | `CaCO₃ → CaO + CO₂` |
| **ऊष्मीय वियोजन** | ऊष्मा | `2FeSO₄ → Fe₂O₃ + SO₂ + SO₃` |
| **ऊष्मीय वियोजन** | ऊष्मा | `2Pb(NO₃)₂ → 2PbO + 4NO₂ + O₂` |
| **प्रकाश-अपघटन** | सूर्य का प्रकाश | `2AgCl → 2Ag + Cl₂` |
| **प्रकाश-अपघटन** | सूर्य का प्रकाश | `2AgBr → 2Ag + Br₂` |
| **वैद्युत अपघटन** | विद्युत | `2H₂O → 2H₂ + O₂` |

> **Note:** AgCl और AgBr का प्रकाश से वियोजन → **श्याम-श्वेत फोटोग्राफी** में उपयोग

#### ❄️ ऊष्माशोषी अभिक्रिया (Endothermic Reaction)

> **परिभाषा:** जिन अभिक्रियाओं में **ऊर्जा अवशोषित** होती है।

वियोजन अभिक्रियाएँ प्रायः ऊष्माशोषी होती हैं क्योंकि अभिकारकों को तोड़ने के लिए ऊष्मा/प्रकाश/विद्युत की जरूरत होती है।

**उदाहरण:** Ba(OH)₂ + NH₄Cl → BaCl₂ + NH₃ + H₂O (परखनली ठंडी हो जाती है)

---

### 🔹 1.2.3 विस्थापन अभिक्रिया (Displacement Reaction)

> **परिभाषा:** जब **एक अधिक क्रियाशील तत्व**, दूसरे **कम क्रियाशील तत्व** को उसके यौगिक से **विस्थापित** कर देता है।

**उदाहरण:**

```
Fe(s) + CuSO₄(aq) → FeSO₄(aq) + Cu(s)   ...(1.24)
Zn(s) + CuSO₄(aq) → ZnSO₄(aq) + Cu(s)   ...(1.25)
Pb(s) + CuCl₂(aq) → PbCl₂(aq) + Cu(s)   ...(1.26)
```

> **क्रियाकलाप 1.9 में:** लोहे की कील → भूरी हो जाती है (Cu जमता है), CuSO₄ का नीला रंग → मलिन हो जाता है।
> 
> **कारण:** Fe, Zn, Pb — Cu से अधिक क्रियाशील हैं।

---

### 🔹 1.2.4 द्विविस्थापन अभिक्रिया (Double Displacement Reaction)

> **परिभाषा:** जिन अभिक्रियाओं में **दो यौगिकों के बीच आयनों का आदान-प्रदान** होता है।

**उदाहरण (अवक्षेपण अभिक्रिया):**

```
Na₂SO₄(aq) + BaCl₂(aq) → BaSO₄(s)↓ + 2NaCl(aq)   ...(1.27)
```

> **अवक्षेप (Precipitate):** जल में अविलेय पदार्थ जो अभिक्रिया में बनता है।
> 
> **अवक्षेपण अभिक्रिया:** जिस अभिक्रिया में अवक्षेप बनता है।

**क्रियाकलाप 1.2 में:**
```
Pb(NO₃)₂(aq) + 2KI(aq) → PbI₂(s)↓ + 2KNO₃(aq)
```
*(PbI₂ = पीले रंग का अवक्षेप)*

---

### 🔹 1.2.5 उपचयन एवं अपचयन / रेडॉक्स अभिक्रिया

| पद | परिभाषा |
|----|---------|
| **उपचयन (Oxidation)** | ऑक्सीजन की **वृद्धि** या हाइड्रोजन का **ह्रास** |
| **अपचयन (Reduction)** | ऑक्सीजन का **ह्रास** या हाइड्रोजन की **वृद्धि** |
| **रेडॉक्स अभिक्रिया** | जब एक पदार्थ उपचयित और दूसरा अपचयित हो |

**उदाहरण:**
```
2Cu(s) + O₂(g) → 2CuO(s)   ...(1.28)  [Cu का उपचयन]

CuO(s) + H₂(g) → Cu(s) + H₂O(l)   ...(1.29)
      ↑ अपचयन        ↑ उपचयन
```

**रेडॉक्स के अन्य उदाहरण:**
```
ZnO + C → Zn + CO   ...(1.31)
  [C उपचयित → CO, ZnO अपचयित → Zn]

MnO₂ + 4HCl → MnCl₂ + 2H₂O + Cl₂   ...(1.32)
  [HCl उपचयित → Cl₂, MnO₂ अपचयित → MnCl₂]
```

## 🌍 1.3 दैनिक जीवन में उपचयन अभिक्रियाओं के प्रभाव

---

### 🔹 1.3.1 संक्षारण (Corrosion)

> **परिभाषा:** जब कोई धातु अम्ल, आर्द्रता आदि के संपर्क में आकर **क्षतिग्रस्त** हो जाती है।

| धातु | संक्षारण |
|------|----------|
| लोहा | लाल-भूरा जंग (Fe₂O₃) |
| चाँदी | काली परत |
| ताँबा | हरी परत |

**हानि:** कार, पुल, रेलिंग, जहाज — बहुत क्षति होती है।

---

### 🔹 1.3.2 विकृतगंधिता (Rancidity)

> **परिभाषा:** तेल/वसायुक्त खाद्य पदार्थ के **उपचयन** से उनके स्वाद और गंध का बदलना।

**रोकने के उपाय:**
- ✅ **प्रति-ऑक्सीकारक** (Antioxidants) मिलाना
- ✅ **वायुरोधी बर्तनों** में खाद्य सामग्री रखना
- ✅ चिप्स की थैली में **नाइट्रोजन गैस** भरना (O₂ हटाकर)

---

## 📊 सभी अभिक्रियाओं का सारांश चार्ट

```
रासायनिक अभिक्रियाएँ
│
├── संयोजन          → A + B → AB
├── वियोजन          → AB → A + B
│   ├── ऊष्मीय
│   ├── प्रकाश
│   └── वैद्युत
├── विस्थापन         → A + BC → AC + B
├── द्विविस्थापन      → AB + CD → AD + CB
│   └── अवक्षेपण
└── रेडॉक्स
    ├── उपचयन
    └── अपचयन
```

```

```

### ✅ 5 प्रकार की अभिक्रियाएँ (एक लाइन में)

| प्रकार | सूत्र | याद रखें |
|--------|-------|----------|

| वियोजन | AB→A+B | तोड़ना |
| विस्थापन | A+BC→AC+B | एक धातु दूसरे को हटाए |
| द्विविस्थापन | AB+CD→AD+CB | आयनों का आदान-प्रदान |
| रेडॉक्स | उपचयन+अपचयन | O₂ बढ़े/घटे |

### ✅ ऊर्जा आधारित अभिक्रियाएँ

| अभिक्रिया | ऊर्जा | उदाहरण |
|-----------|-------|--------|
| ऊष्माक्षेपी | निकलती है 🔥 | दहन, श्वसन, CaO+H₂O |
| ऊष्माशोषी | अवशोषित होती है ❄️ | वियोजन, Ba(OH)₂+NH₄Cl |

### ✅ वियोजन के 3 प्रकार

```
ऊष्मा    → CaCO₃ → CaO + CO₂
प्रकाश   → 2AgCl → 2Ag + Cl₂  (फोटोग्राफी)
विद्युत  → 2H₂O  → 2H₂ + O₂
```

### ✅ उपचयन vs अपचयन

```
उपचयन  = O₂ बढ़े  OR  H₂ घटे
अपचयन  = O₂ घटे  OR  H₂ बढ़े
```

### ✅ महत्वपूर्ण रंग परिवर्तन

| अभिक्रिया | रंग परिवर्तन |
|-----------|-------------|
| CuSO₄ में Fe डालना | नीला → फीका, Fe भूरी हो जाती है |
| FeSO₄ गर्म करना | हरा → लाल-भूरा (Fe₂O₃) |
| AgCl सूरज में | सफेद → धूसर |
| Cu गर्म करना | लाल-भूरा → काला (CuO) |
| Pb(NO₃)₂ गर्म करना | भूरा धुआँ (NO₂) |

### ✅ अवक्षेपण — याद रखें

> **BaSO₄** = सफेद अवक्षेप (जल में अविलेय)
> 
> **PbI₂** = पीला अवक्षेप

### ✅ दैनिक जीवन के उदाहरण

```
संक्षारण   → जंग (Fe), काली परत (Ag), हरी परत (Cu)
विकृतगंधिता → तेल/चिप्स का खराब होना
रोकथाम    → N₂ गैस, वायुरोधी बर्तन, Antioxidant
```

### ✅ भौतिक अवस्था संकेत
```
(s) = Solid | (l) = Liquid | (g) = Gas | (aq) = Aqueous
```

### ✅ TOP 5 परीक्षा उपयोगी प्रश्न

1. **मैग्नीशियम रिबन को जलाने से पहले साफ क्यों करते हैं?**
   → ताकि उस पर चढ़ी MgO की परत हट जाए और शुद्ध Mg जले।

2. **Zn + H₂SO₄ → ZnSO₄ + H₂ — किस प्रकार की अभिक्रिया है?**
   → विस्थापन + रेडॉक्स

3. **जल के वैद्युत अपघटन में H₂ की मात्रा O₂ से दोगुनी क्यों?**
   → 2H₂O → 2H₂ + O₂ — H₂ का अनुपात 2:1 है।

4. **चिप्स की थैली में N₂ क्यों भरी जाती है?**
   → O₂ हटाकर उपचयन (विकृतगंधिता) रोकने के लिए।

5. **लोहे पर पेंट क्यों करते हैं?**
   → O₂ और आर्द्रता से बचाकर संक्षारण (जंग) रोकने के लिए।
