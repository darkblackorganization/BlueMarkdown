# 🧩 PART 1 — रासायनिक अभिक्रिया

## 📌 परिभाषा

> **रासायनिक अभिक्रिया:** जब किसी पदार्थ की **प्रकृति और पहचान बदल जाए** — नया पदार्थ बने — तो उसे रासायनिक अभिक्रिया कहते हैं।

**दैनिक जीवन के उदाहरण:**

| घटना | क्या बदलता है |
|------|--------------|
| दूध का खट्टा होना | नया अम्लीय पदार्थ बनता है |
| लोहे पर जंग | Fe₂O₃ बनता है |
| अंगूर का किण्वन | ग्लूकोज → एथेनॉल बनता है |
| भोजन पकाना | रासायनिक संरचना बदलती है |
| श्वसन | ग्लूकोज + O₂ → CO₂ + H₂O + ऊर्जा |

---

## 📌 रासायनिक अभिक्रिया के लक्षण (4 Indicators)

> **Trick: "अर-रं-गैस-ताप"** — अवस्था, रंग, गैस, तापमान

```
┌─────────────────────────────────────────────┐
│  1️⃣  अवस्था में परिवर्तन  (State Change)       │
│  2️⃣  रंग में परिवर्तन    (Colour Change)       │
│  3️⃣  गैस का निकास       (Gas Evolution)       │
│  4️⃣  तापमान में परिवर्तन (Temperature Change) │
└─────────────────────────────────────────────┘
```

**उदाहरण से समझो:**
- Mg जलाना → **चमकदार सफेद लौ** (रंग), **सफेद राख** (अवस्था) ✅
- Zn + H₂SO₄ → **बुलबुले** (गैस), **परखनली गर्म** (तापमान) ✅
- Pb(NO₃)₂ + KI → **पीला अवक्षेप** (रंग + अवस्था) ✅

# 🧩 PART 2 — रासायनिक समीकरण

## 📌 शब्द-समीकरण (Word Equation)

किसी अभिक्रिया को **शब्दों में** लिखना — सबसे सरल लेकिन लंबा तरीका।

```
मैग्नीशियम  +  ऑक्सीजन  →  मैग्नीशियम ऑक्साइड
 (अभिकारक)    (अभिकारक)        (उत्पाद)
```

> **अभिकारक (Reactants)** = जो भाग लें → **LHS (बाईं ओर)**  
> **उत्पाद (Products)** = जो बनें → **RHS (दाईं ओर)**  
> **तीर (→)** = अभिक्रिया की दिशा

---

## 📌 रासायनिक समीकरण लिखना

शब्दों की जगह **रासायनिक सूत्र** उपयोग करो:

```
शब्द:   मैग्नीशियम + ऑक्सीजन → मैग्नीशियम ऑक्साइड
सूत्र:       Mg     +    O₂    →     MgO          ← ढाँचा/कंकाली समीकरण
```

⚠️ यह **असंतुलित** है — इसे ही **कंकाली समीकरण (Skeleton Equation)** कहते हैं।

---

## 📌 संतुलित रासायनिक समीकरण

### द्रव्यमान संरक्षण का नियम

> **"किसी भी रासायनिक अभिक्रिया में द्रव्यमान का न निर्माण होता है, न विनाश।"**

**मतलब:** LHS के कुल परमाणु = RHS के कुल परमाणु

```
असंतुलित:  Mg + O₂ → MgO
              1    2    1+1=2 ← O असंतुलित!

संतुलित:   2Mg + O₂ → 2MgO  ✅
```

---

### संतुलन की विधि — हिट एंड ट्रायल (Hit and Trial)

**उदाहरण:** `Fe + H₂O → Fe₃O₄ + H₂`

**Step 1:** हर सूत्र को बॉक्स में लो (सूत्र मत बदलो, केवल गुणांक बदलो)

**Step 2:** परमाणु गिनो:

| तत्व | LHS | RHS | संतुलित? |
|------|-----|-----|---------|
| Fe   | 1   | 3   | ❌ |
| H    | 2   | 2   | ✅ |
| O    | 1   | 4   | ❌ |

**Step 3:** सबसे ज़्यादा परमाणु वाले से शुरू करो → Fe₃O₄ → O संतुलित करो
```
Fe + 4H₂O → Fe₃O₄ + H₂
```

**Step 4:** H संतुलित करो
```
Fe + 4H₂O → Fe₃O₄ + 4H₂
```

**Step 5:** Fe संतुलित करो
```
3Fe + 4H₂O → Fe₃O₄ + 4H₂  ✅
```

**Step 6:** जाँच करो:

| तत्व | LHS | RHS |
|------|-----|-----|
| Fe   | 3   | 3 ✅|
| H    | 8   | 8 ✅|
| O    | 4   | 4 ✅|

> ⚠️ **Golden Rule:** `4H₂O` लिख सकते हो — `H₂O₄` या `(H₂O)₄` नहीं! गुणांक बदलो, सूत्र नहीं।

---

## 📌 अवस्था के चिह्न (State Symbols)

| चिह्न | अवस्था | उदाहरण |
|-------|--------|--------|
| **(s)** | Solid (ठोस) | `Fe(s)` |
| **(l)** | Liquid (द्रव) | `H₂O(l)` |
| **(g)** | Gas (गैस) | `CO₂(g)` |
| **(aq)** | Aqueous (जलीय) | `NaCl(aq)` |

> तीर के **ऊपर/नीचे** — ताप, दाब, उत्प्रेरक लिखते हैं।  
> जैसे: `CO(g) + 2H₂(g) →(340atm)→ CH₃OH(l)`

# 🧩 PART 3 — रासायनिक अभिक्रियाओं के प्रकार

---

## 🔵 1. संयोजन अभिक्रिया (Combination Reaction)

> **"दो या दो से अधिक पदार्थ मिलकर → एक नया पदार्थ बनाते हैं"**

```
A  +  B  →  AB
```

### महत्वपूर्ण उदाहरण:

```
CaO(s)  +  H₂O(l)  →  Ca(OH)₂(aq)  +  ऊष्मा
(बिना बुझा चूना)       (बुझा हुआ चूना)

C(s)  +  O₂(g)  →  CO₂(g)    [कोयले का दहन]

2H₂(g)  +  O₂(g)  →  2H₂O(l)  [जल का निर्माण]

2Mg(s)  +  O₂(g)  →  2MgO(s)   [Mg का दहन]
```

**रोचक तथ्य:**
- बुझे चूने (Ca(OH)₂) से दीवार सफेद होती है
- Ca(OH)₂ + CO₂ → CaCO₃ + H₂O → 2-3 दिन बाद **चमक आती है**
- **संगमरमर का सूत्र = CaCO₃** (बुझे चूने जैसा ही!)

---

## 🔴 2. अपघटन / वियोजन अभिक्रिया (Decomposition Reaction)

> **"एक पदार्थ टूटकर → दो या दो से अधिक पदार्थ देता है"**  
> *(संयोजन का बिल्कुल उल्टा)*

```
AB  →  A  +  B
```

### तीन प्रकार:

---

### 🔥 2a. ऊष्मीय अपघटन (Thermal Decomposition)

ऊष्मा देने पर अभिकारक टूटता है।

```
CaCO₃(s)  →(ऊष्मा)→  CaO(s)  +  CO₂(g)
(चूना पत्थर)          (बिना बुझा चूना)

2FeSO₄(s)  →(ऊष्मा)→  Fe₂O₃(s)  +  SO₂(g)  +  SO₃(g)
(हरा रंग)             (लाल-भूरा)

2Pb(NO₃)₂(s)  →(तापन)→  2PbO(s)  +  4NO₂(g)  +  O₂(g)
                          (भूरा धुआँ = NO₂)
```

---

### ⚡ 2b. वैद्युत अपघटन (Electrolytic Decomposition)

विद्युत धारा से अभिकारक टूटता है।

```
2H₂O(l)  →(विद्युत)→  2H₂(g)  +  O₂(g)
```

> **क्रियाकलाप 1.7:** एक परखनली में H₂ दूसरी से **दोगुना** क्यों?  
> क्योंकि 2H₂O → **2H₂** + **1O₂** — H₂ का अनुपात 2:1 है!

---

### ☀️ 2c. प्रकाश अपघटन (Photolytic Decomposition)

सूर्य के प्रकाश से अभिकारक टूटता है।

```
2AgCl(s)  →(सूर्य प्रकाश)→  2Ag(s)  +  Cl₂(g)
(सफेद)                      (धूसर/काला)

2AgBr(s)  →(सूर्य प्रकाश)→  2Ag(s)  +  Br₂(g)
```

> 📸 **AgCl और AgBr** — **श्याम-श्वेत फोटोग्राफी** में उपयोग होता है!

---

## 🟠 3. विस्थापन अभिक्रिया (Displacement Reaction)

> **"अधिक क्रियाशील तत्व → कम क्रियाशील तत्व को उसके यौगिक से बाहर करता है"**

```
A  +  BC  →  AC  +  B
(अधिक क्रियाशील)   (विस्थापित)
```

### महत्वपूर्ण उदाहरण:

```
Fe(s)  +  CuSO₄(aq)  →  FeSO₄(aq)  +  Cu(s)
Zn(s)  +  CuSO₄(aq)  →  ZnSO₄(aq)  +  Cu(s)
Pb(s)  +  CuCl₂(aq)  →  PbCl₂(aq)  +  Cu(s)
```

### क्रियाकलाप 1.9 में क्या हुआ?

| बदलाव | कारण |
|-------|------|
| लोहे की कील **भूरी** हो गई | Cu जम गया कील पर |
| CuSO₄ का **नीला रंग फीका** पड़ा | Cu²⁺ आयन कम हुए |

> **याद रखो क्रियाशीलता:** Zn > Fe > Pb > **Cu** (Cu सबसे कम!)

---

## 🟣 4. द्विविस्थापन अभिक्रिया (Double Displacement Reaction)

> **"दो यौगिकों के बीच आयनों का आदान-प्रदान होता है"**

```
AB  +  CD  →  AD  +  CB
```

### 🔸 4a. अवक्षेपण अभिक्रिया (Precipitation Reaction)

> जिसमें **जल में अविलेय ठोस (अवक्षेप)** बनता है।

```
Na₂SO₄(aq)  +  BaCl₂(aq)  →  BaSO₄(s)↓  +  2NaCl(aq)
                               (सफेद अवक्षेप)

Pb(NO₃)₂(aq)  +  2KI(aq)  →  PbI₂(s)↓  +  2KNO₃(aq)
                               (पीला अवक्षेप)
```

### 🔸 4b. तटस्थीकरण अभिक्रिया (Neutralisation Reaction)

> **अम्ल + क्षार → लवण + जल**

```
NaOH(aq)  +  HCl(aq)  →  NaCl(aq)  +  H₂O(l)
```

> यह भी **द्विविस्थापन** है — Na⁺ और H⁺ का आदान-प्रदान!

# 🧩 PART 4 — ऑक्सीकरण एवं अपचयन

## 📌 परिभाषाएँ

| पद | ऑक्सीजन के आधार पर | हाइड्रोजन के आधार पर |
|----|--------------------|---------------------|
| **ऑक्सीकरण / उपचयन** | O₂ की **वृद्धि** | H₂ का **ह्रास** |
| **अपचयन (Reduction)** | O₂ का **ह्रास** | H₂ की **वृद्धि** |

> **Trick:** OIL RIG  
> **O**xidation **I**s **L**oss — **R**eduction **I**s **G**ain  
> (यहाँ electrons की बात है, लेकिन O₂/H₂ पर भी लागू होती है)

---

## 📌 रेडॉक्स अभिक्रिया (Redox Reaction)

> जिस अभिक्रिया में **एक पदार्थ उपचयित** और **दूसरा अपचयित** हो।

```
        उपचयन ↓
CuO(s) + H₂(g) →(तापन)→ Cu(s) + H₂O(l)
        अपचयन ↑
```

**विश्लेषण:**
- CuO में O₂ का ह्रास → **CuO अपचयित**
- H₂ में O₂ की वृद्धि → **H₂ उपचयित**

### और उदाहरण:

```
ZnO + C → Zn + CO
→ C उपचयित (CO बना), ZnO अपचयित (Zn बना)

MnO₂ + 4HCl → MnCl₂ + 2H₂O + Cl₂
→ HCl उपचयित (Cl₂ बना), MnO₂ अपचयित (MnCl₂ बना)

2Cu + O₂ →(तापन)→ 2CuO
→ Cu उपचयित (O₂ जुड़ी)
```

# 🧩 PART 5 — ऊर्जा परिवर्तन

## 🔥 ऊष्माक्षेपी अभिक्रिया (Exothermic Reaction)

> **"उत्पाद बनने के साथ ऊष्मा भी निकलती है"**

```
┌──────────────────────────────────────────────┐
│  ऊर्जा  →  बाहर निकलती है  → वातावरण गर्म  │
└──────────────────────────────────────────────┘
```

| उदाहरण | समीकरण |
|---------|--------|
| प्राकृतिक गैस दहन | `CH₄ + 2O₂ → CO₂ + 2H₂O + ऊर्जा` |
| CaO + जल | `CaO + H₂O → Ca(OH)₂ + ऊष्मा` |
| **श्वसन** | `C₆H₁₂O₆ + 6O₂ → 6CO₂ + 6H₂O + ऊर्जा` |
| कंपोस्ट बनना | सब्जियों का विघटन |
| Mg का दहन | `2Mg + O₂ → 2MgO + ऊष्मा` |

> ⭐ **श्वसन ऊष्माक्षेपी है** — हमारे शरीर को इसी से ऊर्जा मिलती है!

---

## ❄️ ऊष्माशोषी अभिक्रिया (Endothermic Reaction)

> **"अभिक्रिया होने के लिए ऊर्जा अवशोषित (अंदर जाती) होती है"**

```
┌──────────────────────────────────────────────┐
│  ऊर्जा  →  अंदर जाती है  → वातावरण ठंडा    │
└──────────────────────────────────────────────┘
```

| उदाहरण | कारण |
|---------|------|
| Ba(OH)₂ + NH₄Cl → BaCl₂ + NH₃ + H₂O | परखनली ठंडी होती है |
| CaCO₃ → CaO + CO₂ (ऊष्मा चाहिए) | ऊष्मा देने पर ही होता है |
| वैद्युत अपघटन | विद्युत ऊर्जा चाहिए |
| प्रकाश अपघटन | प्रकाश ऊर्जा चाहिए |

> **Logic:** सभी वियोजन अभिक्रियाएँ प्रायः **ऊष्माशोषी** होती हैं — तोड़ने के लिए ऊर्जा चाहिए!

# 🧩 PART 6 — ऑक्सीकरण के प्रभाव

---

## 🦠 जंग लगना (Rusting)

> लोहे पर O₂ और H₂O की उपस्थिति में **लाल-भूरी परत** चढ़ना।

```
4Fe + 3O₂ + xH₂O → 2Fe₂O₃.xH₂O
                    (जंग / Rust)
```

---

## 🔩 संक्षारण (Corrosion)

> जब कोई **धातु** अम्ल, आर्द्रता आदि के संपर्क में आकर **क्षतिग्रस्त** हो।

| धातु | संक्षारण का रंग |
|------|----------------|
| लोहा (Fe) | लाल-भूरा (जंग) |
| चाँदी (Ag) | काली परत |
| ताँबा (Cu) | हरी परत |

**हानि:** कार, पुल, रेलिंग, जहाज — भारी आर्थिक नुकसान।

---

### 🛡️ संक्षारण से बचाव

```
1. पेंट करना          → O₂ और नमी से बचाता है
2. ग्रीस/तेल लगाना    → धातु को ढकता है
3. गैल्वेनाइजिंग      → Zn की परत चढ़ाना
4. मिश्रधातु बनाना    → जैसे स्टेनलेस स्टील
5. विद्युत लेपन       → Cr/Ni की परत
```

---

## 🤢 विकृतगंधिता (Rancidity)

> तेल/वसायुक्त खाद्य पदार्थ का **उपचयन** होने से उसके **स्वाद और गंध का बदलना।**

**कारण:** O₂ तेल/वसा से क्रिया करता है → नए यौगिक बनते हैं → बदबू/खट्टापन आता है।

---

### 🛡️ विकृतगंधिता से बचाव

| उपाय | कैसे काम करता है |
|------|-----------------|
| **प्रति-ऑक्सीकारक** मिलाना | O₂ की क्रिया रोकता है |
| **वायुरोधी बर्तन** में रखना | O₂ से संपर्क नहीं होता |
| **नाइट्रोजन गैस** भरना | O₂ हट जाती है, N₂ निष्क्रिय है |
| **फ्रिज में रखना** | कम तापमान पर उपचयन धीमा |

> 🍟 **चिप्स की थैली में N₂** इसीलिए भरी जाती है!

| द्विविस्थापन | AB+CD→AD+CB | आयन exchange | Na₂SO₄+BaCl₂→BaSO₄+NaCl |
| रेडॉक्स | O₂ का आदान-प्रदान | उपचयन+अपचयन साथ | CuO+H₂→Cu+H₂O |
| ऊष्माक्षेपी | ऊष्मा निकले | गर्माहट महसूस | दहन, श्वसन |
| ऊष्माशोषी | ऊष्मा अवशोषित | ठंडक महसूस | वियोजन |

# 📋 सभी महत्वपूर्ण समीकरण — एक जगह

```
संयोजन:
  2Mg + O₂ → 2MgO
  CaO + H₂O → Ca(OH)₂ + ऊष्मा
  Ca(OH)₂ + CO₂ → CaCO₃ + H₂O
  2H₂ + O₂ → 2H₂O
  C + O₂ → CO₂
  CH₄ + 2O₂ → CO₂ + 2H₂O + ऊर्जा

अपघटन:
  CaCO₃ →(ऊष्मा)→ CaO + CO₂
  2FeSO₄ →(ऊष्मा)→ Fe₂O₃ + SO₂ + SO₃
  2Pb(NO₃)₂ →(ऊष्मा)→ 2PbO + 4NO₂ + O₂
  2AgCl →(सूर्य)→ 2Ag + Cl₂
  2AgBr →(सूर्य)→ 2Ag + Br₂
  2H₂O →(विद्युत)→ 2H₂ + O₂

विस्थापन:
  Fe + CuSO₄ → FeSO₄ + Cu
  Zn + CuSO₄ → ZnSO₄ + Cu
  Pb + CuCl₂ → PbCl₂ + Cu

द्विविस्थापन:
  Na₂SO₄ + BaCl₂ → BaSO₄↓ + 2NaCl
  Pb(NO₃)₂ + 2KI → PbI₂↓ + 2KNO₃
  NaOH + HCl → NaCl + H₂O

रेडॉक्स:
  CuO + H₂ → Cu + H₂O
  ZnO + C → Zn + CO
  MnO₂ + 4HCl → MnCl₂ + 2H₂O + Cl₂
  2Cu + O₂ → 2CuO

संतुलित (महत्वपूर्ण):
  Zn + H₂SO₄ → ZnSO₄ + H₂
  3Fe + 4H₂O → Fe₃O₄ + 4H₂
  C₆H₁₂O₆ + 6O₂ → 6CO₂ + 6H₂O + ऊर्जा
```

## 🎯 2 — अभिक्रियाओं की पहचान (Shortcut Logic)

```
✅ एक उत्पाद बना?          → संयोजन
✅ एक अभिकारक टूटा?        → अपघटन
✅ एक धातु ने दूसरे को हटाया? → विस्थापन
✅ दो यौगिक बदले?           → द्विविस्थापन
✅ अवक्षेप बना?              → अवक्षेपण (द्विविस्थापन का भाग)
✅ O₂ बढ़ी + घटी दोनों?      → रेडॉक्स
✅ गर्मी निकली?              → ऊष्माक्षेपी
✅ ऊर्जा लगी?               → ऊष्माशोषी
```

## 🎯 3 — उपचयन vs अपचयन (1 line में)

```
उपचयन  = O₂ बढ़े  या  H₂ घटे   → "ऑक्सी मिला, Hydrogen गया"
अपचयन  = O₂ घटे  या  H₂ बढ़े   → "ऑक्सी गई, Hydrogen मिला"
```

## 🎯 8 — TOP 7 परीक्षा प्रश्न + उत्तर

**Q1. Mg को जलाने से पहले रेगमाल से क्यों साफ करते हैं?**
> ताकि उस पर चढ़ी MgO की परत हट जाए और शुद्ध Mg जले।

**Q2. वैद्युत अपघटन में H₂ दोगुना क्यों?**
> 2H₂O → **2H₂** + O₂ → H₂:O₂ = 2:1

**Q3. श्वसन ऊष्माक्षेपी क्यों है?**
> ग्लूकोज + O₂ → CO₂ + H₂O + ऊर्जा → ऊर्जा निकलती है → शरीर गर्म रहता है।

**Q4. चिप्स में N₂ क्यों?**
> O₂ हटाकर उपचयन (विकृतगंधिता) रोकने के लिए।

**Q5. लोहे को पेंट क्यों करते हैं?**
> O₂ और नमी से बचाकर संक्षारण (जंग) रोकने के लिए।

**Q6. AgCl फोटोग्राफी में क्यों उपयोगी है?**
> सूर्य प्रकाश में AgCl → Ag + Cl₂ — रोशनी के अनुसार Ag जमता है → फोटो बनती है।

**Q7. वियोजन को संयोजन का विपरीत क्यों कहते हैं?**
> संयोजन: A+B→AB (जोड़ना) | वियोजन: AB→A+B (तोड़ना) — बिल्कुल उल्टी प्रक्रिया।
